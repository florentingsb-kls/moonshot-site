DOC MANAGER ANTICIPÉ

Il s’agit des docs managers validés en anticipés, hors doc manager et défaut de communication.

Ces docs concernent uniquement ceux relatifs aux installations réalisées sur la dernière semaine de clôture d’installation. La demande doit être effectuée par mail à contactST@verisure.fr par l’assistante d’agence, le chef d’agence et le territorial manager, en précisant une date d’intervention convenue avec le client.

Si la demande est conforme aux règles d’éligibilité, alors le master devra lever le doc manager et ouvrir un com log sur l’installation du client. Le com log ne sera clôturé qu’une fois que l’ES aura réglé définitivement le doc manager chez le client.

Dans le cas où la demande n’est pas conforme, l’ES devra intervenir pour régler le doc manager chez le client.
