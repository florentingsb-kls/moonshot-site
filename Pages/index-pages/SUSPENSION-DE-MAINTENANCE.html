LES SUSPENSIONS DE MAINTENANCES :

Dans certaines situations exceptionnelles, un technicien peut être amené à contacter le support technique afin de demander une sortie de test. Voici une liste des différents cas :
  • Le client doit quitter les lieux
  • Le technicien n’a pas le matériel en stock
  • Urgence personnelle inattendue pour le technicien
  • Le technicien rencontre des difficultés pour dépanner le client

Pour ce genre de situations, le collaborateur du support technique doit effectuer les actions suivantes :

  • S’assurer que la centrale est bien configurée en CCS.
  • Vérifier la transmission en demandant un SOS clavier.
  • Faire la sortie de test.
  • Cocher la case acti BAC et renseigner les infos dans inst.comp
  • Laisser une note dans SBN pour expliquer la raison de la sortie test
  • Le technicien doit contacter le planning.
